Technologies Used

Power Automate

Automated alerts and report distribution to stakeholders via email and Teams.

The Challenge



A manufacturing company with multiple production lines was relying on manual data collection and Excel-based reporting to track KPIs like OEE (Overall Equipment Effectiveness), scrap rates, and production throughput. Reports were generated weekly by a dedicated analyst, often arriving too late to act on emerging quality issues. Management needed real-time visibility into production performance.

The Solution



I built an end-to-end automated reporting pipeline using Azure Data Factory to ingest data from shop floor systems, ERP, and IoT sensors on a scheduled basis. The data was transformed and loaded into a SQL-based data warehouse with optimized star schemas for analytical queries. Power BI dashboards provided real-time production monitoring, and Power Automate triggered alerts when KPIs fell below threshold values, enabling immediate corrective action.

Business Impact



20+ Hours Saved

Weekly manual reporting effort eliminated through fully automated data pipelines.

Real-Time Monitoring

Production managers gained live dashboards replacing weekly static Excel reports.

Proactive Alerts

Automated threshold alerts reduced quality issue response time from days to minutes.
