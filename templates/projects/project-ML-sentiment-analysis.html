ML News Article Sentiment


Machine Learning / Data Science


Technology Used: ⚡ ML | ⚡ Python | ⚡ Docker

The Problem:

Sometimes it is important for algorithms such as trading bots to quickly determine the sentiment of a news article. This task can be done manually, however, it is very time-consuming and tedious. Hence, this machine learning algorithm set out to solve this problem by pre-training a model on a large dataset of news articles from various sources.

The Solution:

Create a machine learning algorithm that can quickly determine the sentiment of a news article. The algorithm was trained on a large dataset of news articles from various sources. The algorithm is fully dynamic and automatically updates with every new file upload. Furthermore, the user can choose between different types of analysis or analyze all data at once.
