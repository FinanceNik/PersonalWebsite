Duolingo Analysis Tool


Data Analysis / Data Visualization


Technology Used: ⚡ Python | ⚡ Google Cloud Deploy | ⚡ Data Analysis

The Problem:

Duolingo is a language-learning platform that includes a language-learning website and app, as well as a digital language proficiency assessment exam. Duolingo offers all its language courses free of charge. As of November 2019, the language-learning website and app offer 90 different language courses in 22 languages. The app has over 300 million registered users across the world. I have been using Duolingo for over 5 years now and have completed over 1000 lessons. However, I was always curious about my learning progress and wanted to analyze my learning behavior. Unfortunately, Duolingo does not offer any data analysis tools for its users. Hence, I set out to create my own data analysis tool for Duolingo.

The Solution:

The goal was to create a tool that makes it easy to analyze your Duolingo learning behavior. The tool is a web application that is hosted on Google Cloud and can be accessed from anywhere. The tool is fully responsive and can be used devices with any screen size. The tool is fully dynamic and automatically updates with every new file upload. Furthermore, the user can choose between different types of analysis or analyze all data at once.

Key Features:

The dashboard has a file upload feature. With that every user can upload their own Duolingo data and get a personalized dashboard of their own. The dashboard is fully dynamic and automatically updates with every new file upload. Furthermore, the user can choose between different types of analysis or analyze all data at once.


  • File upload interface
  • Data analysis on your data
  • Dynamic dashboard
  • Visualization of personalized data
