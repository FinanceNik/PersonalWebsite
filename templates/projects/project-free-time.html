Software Development / Process Automation


Technology Used: ⚡ Python | ⚡ Google Cloud Deploy | ⚡ Process Automation

The Problem:

As a freelancer, more often than not, you have to proof your exact working hours to your clients. This is frequently done using various Excel sheets and templates. This can be a very time-consuming and tedious and hence I set out to create a tool that makes it easier to track time as well as creating invoices for clients.

The Solution:

The goal was to create a tool that makes it easy to track time and create invoices for clients, thereby reducing the time freelancers have to spend on administrative tasks. The tool is a web application that is hosted on Google Cloud and can be accessed from anywhere. The tool is fully responsive and can be used devices with any screen size.

Key Features:

The application allows the user to enter the working hours for each day and then automatically calculates the invoice amounts based on the users settings. Afterward, the user can specify the invoice duration and number and export the invoice as a PDF file. The application also allows the user to export the data as a CSV file.


  • Data entry terminal
  • Storing of data in SQL database
  • Export of invoices

Summary:

The application is a great tool for freelancers to track their working hours and create invoices for clients. The application is fully responsive and can be used on any device. The application is hosted on Google Cloud and can be accessed from anywhere. The application is fully dynamic and automatically updates with every new file upload. Furthermore, the user can choose between different types of analysis or analyze all data at once.
