Personal Website


Creating APIs / Designing Websites / Server Deployment


Technology Used: ⚡ Python | ⚡ Google Cloud Deploy | ⚡ HTML / CSS | ⚡ Docker

The Problem:

Of course, one could simply use a website builder like Wix or Squarespace to create a personal website. However, I wanted to create a website that is fully customizable and that I can change and adapt to my needs. Furthermore, I wanted to get up to speed with the latest HTML / CSS technologies and learn how to deploy a website on a Raspeberry Pi.

The Solution:

I created this website from scratch using HTML and CSS. I used the Flask framework to deploy the website on a Raspberry Pi. The website is fully responsive and can be viewed on any device. The methodology of this website is the so-called mobile first approach. This means that the website is first designed for mobile devices and then for larger devices. I incorporated the latest HTML and CSS technologies such as CSS Grid and Flexbox, animations and transitions as well as much more.

Key Features:

The main goal of this website is to showcase my data science projects and to provide a way for potential clients to contact me. The call-to-action is the contact button in the navigation bar and various other elements of the website. Moreover, the projects section shows a selection of my data science and process automation projects. The contact-form triggers a CRUD framework that saves the data in a database. The data can then be accessed via a REST API that is also written in Python's flask framework.

Summary:

In summary, this project was a great way to get up to speed with the latest HTML and CSS technologies. Additionally, I learned how to deploy a website on a Raspberry Pi and saved a little bit of continuous costs by not having to pay for a website builder or hosting service.
